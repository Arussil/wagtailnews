.. _installing:

=====
Setup
=====

``wagtailnews`` is compatible with Wagtail 0.7 and higher,
and Python 2.7 and Python 3.4 and higher.

Step 1
______

Install ``wagtailnews`` using pip::

   pip install wagtailnews

Step 2
______

Add ``wagtailnews`` to installed apps in settings:

.. code-block:: python

  INSTALLED_APPS = [
      'yourproject',
      'wagtailnews',
  ]
